# MSAL

MS Graph Authentication Library

## Installation

### On the system

```ruby
gem install msal
```

### On your project

Add this line to your application's Gemfile:

```ruby
gem 'msal', '~> 0.14.2'
```

And then execute:

    $ bundle install

Or install it yourself as:

    $ gem install msal
